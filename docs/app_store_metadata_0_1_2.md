## Description Draft
Power6 helps you choose the six tasks that matter most each day, complete them with focus, and build momentum from real progress.

Instead of managing an endless task list, Power6 keeps your daily plan intentionally limited. Add your six focus tasks, track completion, review what is still open, and build consistency with streaks and badges.

Power6 is built for busy professionals, freelancers, students, and anyone who wants a simple daily execution system.

Core features:
- Six-task daily planning
- Quick task entry and completion tracking
- Review flow for unfinished work
- Streak and badge motivation
- Timeline history
- Pro insights and CSV export
- Plus, Pro, and Elite subscription tiers

Use Power6 to make each day clearer, lighter, and easier to finish.
