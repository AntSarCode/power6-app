Guideline 5.1.1 - Account Deletion

Power6 now supports in-app account deletion. A signed-in user can delete their account directly inside the app without being sent to a website or customer support flow.

Account deletion path:
Menu > Account settings > Delete Account

The in-app deletion flow:
1. User signs in.
2. User opens Menu.
3. User selects Account settings.
4. User taps Delete Account.
5. User confirms the deletion dialog.
6. The app deletes the account and associated Power6 data, including tasks, subscription records, badges, and related account data.
7. The user is logged out and returned to the login screen.

A physical-device screen recording demonstrating this flow has been attached with this reply/submission.
